COLOURS IN CSS

Colors in CSS

-CSS supports up to 140 colors

-We can add a colored background, for this we have to do it inside the "body" tag. "Ex: background-color: blue" (Background means background.)

-To add color to a text we put it inside the font tag. Ex: font color = "white"

RGB

-The RGB is the treatment of the video signal that treats the signals of the three colors red, green and blue separately. Used independently, it provides higher quality and more faithful color reproduction.

How to use Alpha

It should be clear that Alpha refers to its opacity. 0 would be totally transparent, 1 would be totally opaque and 0.5 would be 50% transparency Example: rgba (255, 125, 0, 0.5)
